 lógica básica de SpotiDante con POO en Java

Se implementa la lógica principal de SpotiDante utilizando:
- Constructores para crear objetos
- Getters y setters para encapsulación
- Lógica de programación en Java

Este proyecto corresponde a ejercicios de la escuela Alura
y puede ser modificado para prácticas y mejoras futuras.
